# 3D Island Timeline - Interactive Web Application

## Project Overview

This project is an interactive 3D web application built with Node.js, Docker, and Three.js. It features a floating 3D island with a character that can be controlled by the user to explore a timeline of information points distributed around the island.

The application is currently running in a Docker container and accessible via a web browser at localhost:3000/island. The interface includes a navigation menu with Home, About, Contact, and 3D Island sections, and the 3D scene displays a floating island with trees, houses, and rocks, along with a controllable character.

## Current Implementation Status

As seen in the screenshots, the application is functional with:
- A working 3D scene showing the island environment
- Navigation controls for camera movement (mouse rotation, zooming)
- Character movement using arrow keys
- Instruction panels that appear and fade out after 5 seconds
- Timeline markers placed around the island

## Key Components

### 3D Island
The island is created using Three.js geometric primitives:
- Cylindrical base with a radius of 15 units
- Grass layer on top
- Randomly placed trees (using cylinder and cone geometries)
- Houses (using box and cone geometries)
- Rocks (using dodecahedron geometry)
- Water plane surrounding the island
- Gentle floating animation that moves the island up and down

### Character
A simple 3D character made of geometric primitives:
- Cylindrical body
- Spherical head
- Cylindrical arms and legs
- Walking animation when moving
- Controlled with arrow keys
- Constrained to stay on the island

### Timeline System
- Eight information points around the island perimeter
- Red cylindrical markers indicate timeline stops
- Information panel appears when character approaches a marker
- Each stop contains a title and descriptive content about a specific timeline event

### User Interface
- Navigation menu (Home, About, Contact, 3D Island)
- Instruction panel that fades out after 5 seconds
- Information panel for timeline content
- Mobile control buttons for touch devices

### Key Code Sections

#### Scene Initialization
```javascript
const scene = new THREE.Scene();
scene.background = new THREE.Color(0x87CEEB); // Sky blue background

const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
camera.position.set(0, 15, 25);

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(window.innerWidth, window.innerHeight);
renderer.setPixelRatio(window.devicePixelRatio);
renderer.shadowMap.enabled = true;
document.getElementById('scene-container').appendChild(renderer.domElement);
```

#### Island Creation
```javascript
const islandRadius = 15;
const islandGeometry = new THREE.CylinderGeometry(islandRadius, islandRadius * 0.8, 3, 64);
const islandMaterial = new THREE.MeshPhongMaterial({ color: 0x8B4513 }); // Brown
const island = new THREE.Mesh(islandGeometry, islandMaterial);
island.castShadow = true;
island.receiveShadow = true;
scene.add(island);

// Add grass on top
const grassGeometry = new THREE.CylinderGeometry(islandRadius, islandRadius, 0.5, 64);
const grassMaterial = new THREE.MeshPhongMaterial({ color: 0x7CFC00 }); // Lawn green
const grass = new THREE.Mesh(grassGeometry, grassMaterial);
grass.position.y = 1.75; // Adjusted for taller island
grass.castShadow = true;
grass.receiveShadow = true;
scene.add(grass);
```

#### Character Movement
```javascript
// Character movement variables
let moveForward = false;
let moveBackward = false;
let turnLeft = false;
let turnRight = false;
const moveSpeed = 0.1;
const turnSpeed = 0.05;

// Update character position and rotation based on arrow key input
if (moveForward) {
    // Move in the direction the character is facing
    character.position.x += Math.sin(character.rotation.y) * moveSpeed;
    character.position.z += Math.cos(character.rotation.y) * moveSpeed;
    
    // Animate legs while walking
    const walkCycle = Math.sin(Date.now() * 0.01) * 0.2;
    character.children[4].rotation.x = walkCycle; // Left leg
    character.children[5].rotation.x = -walkCycle; // Right leg
    character.children[2].rotation.x = -walkCycle * 0.5; // Left arm
    character.children[3].rotation.x = walkCycle * 0.5; // Right arm
}
```

#### Timeline Information Display
```javascript
// Function to display information at current position
function displayNearestStopInfo() {
    // Find the nearest marker
    let nearestIndex = 0;
    let minDistance = Infinity;
    
    markers.forEach((marker, index) => {
        const distance = character.position.distanceTo(marker.position);
        if (distance < minDistance) {
            minDistance = distance;
            nearestIndex = index;
        }
    });
    
    // If close enough to a marker, show info
    if (minDistance < 1) {
        const stop = timelineStops[nearestIndex];
        infoPanel.innerHTML = `<h3>${stop.title}</h3><p>${stop.content}</p>`;
        infoPanel.style.display = 'block';
    } else {
        infoPanel.style.display = 'none';
    }
}
```

## Docker Setup

The application is containerized using Docker with the following Dockerfile:

```dockerfile
FROM node:14

WORKDIR /app

COPY package*.json ./

RUN npm install

COPY . .

EXPOSE 3000

CMD ["node", "app.js"]
```

To build and run the container:
```bash
docker build -t node-website .
docker run -p 3000:3000 -d --name my-node-website node-website
```

## How to Use

1. **Navigation**:
   - Use arrow keys to move the character forward/backward
   - Use left/right arrow keys to rotate the character
   - Click and drag to rotate the camera view
   - Right-click and drag to pan the camera
   - Scroll to zoom in/out

2. **Timeline Exploration**:
   - Navigate the character to the red markers around the island
   - When close to a marker, an information panel will appear
   - Read the timeline information at each stop

3. **Mobile Controls**:
   - Use the on-screen arrow buttons to move the character
   - Touch and drag to control the camera

## Troubleshooting

Common issues and solutions:

- **Black screen**: Check browser console for errors, ensure WebGL is supported
- **Missing 3D elements**: Verify all script paths are correct
- **Performance issues**: Reduce the number of objects or their complexity
- **Docker issues**: Check container logs with `docker logs my-node-website`
- **Character movement problems**: Ensure keyboard event listeners are properly set up
